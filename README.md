# task-1---cybersecurity

Network Scanning and Security Analysis using Nmap and Wireshark

As part of this task, I installed and used Nmap to analyze my local network. I first identified the local IP address range and then performed a TCP SYN scan using the command "nmap -sS" to detect active devices and identify open ports on the network.

After completing the scan, I observed the open ports available on the device. I also used Wireshark to observe and analyze network packets, which helped me better understand how devices communicate within the network.

In addition, I researched the common services running on the detected ports and examined possible security risks related to exposed ports and network services. Finally, I saved the scan results in a text file.
